# Oscillating Speaker Synth
## A PCB design that creates a synth-like sound with an oscillating circuit.

I wanted to create something like a Stylophone that creates a synth sound, but with a simple circuit embedded in a PCB, with only pitch and volume control. I figured I could create some interesting sounds and play around with making multiple of them. I plan to expand on this project with more synth controllers, but I want to start with this one to get a basic understanding of synth circuits, PCB design, and Hack Club's YSWS opportunities.
